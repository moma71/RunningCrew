🎤 우리도 뛴다

Episode 18: 대회 당일 (전반부)

🏃 The Novel Section

1. 출발, 오전 8시

"출발!!!"

수천 명의 러너들이 일제히 뛰기 시작했다.

우리도뛴다 크루 13명도 출발했다.

"가자!!!"

진우가 환호했다.

인파가 밀렸다. 사람들이 빽빽했다.

"어우, 사람 많다..."

순옥이 중얼거렸다.

민수는 Strava를 켰다. 실시간 트래킹이 시작됐다.

[출발 - 0km]

2. 1-3km 지점, 오전 8시 10분

인파가 흩어졌다. 빠른 조가 앞서 나갔다.

"민수 형! 먼저 갈게요!"

유미와 태수는 천천히 뛰었다.

[유미 - 1.0km]

심박수: 148 bpm

3km 지점. 유미가 숨을 헐떡였다.

[유미 - 3.0km]

심박수: 165 bpm

"...쉬어도 돼."

"아니요... 가야죠..."

주변 러너들이 지나쳤다.

"...속도 보지 마. ...네 페이스."

3. 5-6km 지점, 오전 9시

민수가 5km 통과.

[민수 - 5.0km]

심박수: 158 bpm

'절반... 할 수 있어...'

재희는 무릎이 욱신거렸다.

"재희야, 괜찮아?"

"응... 그냥 좀..."

'참아야 해...'

순옥도 힘들어했다.

[순옥 - 6.0km]

심박수: 172 bpm

"아이고메... 죽겄다..."

"물 좀..."

석철이 물을 건넸다.

4. 7km 지점, 오전 9시 50분

유미는 7km 표지판을 봤다.

[유미 - 7.0km]

- 심박수: 170 bpm

- 페이스: 9분 40초/km

'7km... 3km만 더...'

하지만 다리가 무거웠다.

'지난번엔... 8km에서 멈췄는데...'

"형... 저... 무서워요..."

"...뭐가?"

"...또 못 할까봐..."

태수는 유미를 쳐다봤다.

"...이번엔 다르잖아."

"네?"

"...대회야. ...같이 뛰잖아. ...우리 다 있어."

유미는 주변을 봤다.

앞에 민수, 재희, 민지가 뛰고 있었다.

뒤에서 순옥, 석철도 따라오고 있었다.

'...맞아. 혼자 아니야.'

"...고마워요, 형."

5. 8km 지점, 오전 10시 20분

유미가 8km 표지판에 도착했다.

[유미 - 8.0km]

- 심박수: 175 bpm

- 페이스: 10분 10초/km

'여기... 지난번에 멈춘 곳...'

유미는 갑자기 다리에 힘이 빠졌다.

"으..."

주저앉았다.

"형... 못 하겠어요..."

태수가 멈춰 섰다.

"...왜?"

"...다리가... 안 움직여요... 너무 무거워요..."

유미가 울기 시작했다.

"...또 실패했어요... 저 안 돼요..."

6. 크루원들 멈춤, 같은 시각

민수가 뒤를 돌아봤다.

유미가 8km 지점에 주저앉아 있었다.

"유미야!"

민수가 달려갔다.

재희도 멈춰 섰다.

"유미야!"

순옥도 뛰어왔다.

"유미야, 왜 그래!"

석철, 민지, 소희, 수정... 크루원들이 하나둘 모여들었다.

빠른 조에서 뛰던 진우와 동현도 돌아왔다.

"유미 누나!"

크루원 13명 전원이 유미 주위에 모였다.

7. 2km 남음, 오전 10시 25분

"유미야, 괜찮아?"

민수가 물었다.

"민수 형... 저... 못 하겠어요... 다리가 안 움직여요..."

"유미야, 2km만 더야. 할 수 있어."

재희가 말했다.

"못 해요... 저 지난번에도 8km에서..."

순옥이 유미를 안았다.

"유미야, 우리 다 같이 가자. 천천히."

"하지만..."

진우가 말했다.

"유미 누나, 우리 다 같이 뛸게요! 같이 완주해요!"

동현이 말했다.

"유미야, 우리가 옆에 있어. 걱정 마."

태수가 유미의 손을 잡았다.

"...일어나. ...같이 가자."

유미는 눈물을 흘리며 고개를 들었다.

크루원 13명이 유미를 둘러싸고 있었다.

"...정말... 같이 가요?"

"당연하제!"

순옥이 웃었다.

"우리 가족 아이가!"

유미는 눈물을 닦았다.

"...감사합니다... 다들..."

8. 함께 출발, 오전 10시 30분

유미가 일어났다.

태수가 오른쪽, 민수가 왼쪽에서 유미를 지지했다.

"자, 천천히 가자."

"2km만 더야!"

크루원 13명이 유미를 중심으로 느린 페이스로 뛰기 시작했다.

빠른 조도, 중간 조도, 느린 조도 없었다.

우리도뛴다 크루 전원이 함께 뛰었다.

"유미야, 할 수 있어!"

"조금만 더!"

"우리 같이 가자!"

유미는 울면서 뛰었다.

'나... 혼자가 아니야...'

🏃 대회 실시간 트래킹 앱 활용하기

대규모 러닝 대회에서는 실시간 트래킹 앱으로 본인과 동료의 위치를 확인할 수 있습니다.

1️⃣ Logic: 왜 실시간 트래킹?

안전과 응원:

- 참가자 위치 파악

- 가족/친구가 실시간 응원

- 응급 상황 대처

- 페이스 조절 참고

주요 대회 앱:

- 서울 마라톤: 공식 앱

- 춘천 마라톤: 실시간 순위

- 대부분 GPS 기반

2️⃣ Stack: 필요한 것

- 대회 공식 앱 (사전 다운로드)

- 배번호 등록

- GPS 활성화

- 데이터 또는 Wi-Fi

3️⃣ Guide: 3단계 활용

Step 1: 사전 등록

1. 대회 앱 다운로드 2. 배번호 입력 3. 프로필 설정 4. 위치 권한 허용

Step 2: 출발 전 확인

1. 앱 실행 2. GPS 신호 확인 (녹색) 3. "트래킹 시작" 활성화 4. 배터리 충분한지 확인

Step 3: 런닝 중

화면 표시: - 현재 위치 (지도) - 통과 지점 시간 - 예상 완주 시간 - 다른 참가자 위치 (선택)

4️⃣ Prompt: 실전 활용

가족/친구 응원:

"배번호 12345 입력 → 실시간 위치 확인 → 5km 지점 도착 알림"

크루 위치 확인:

"같은 크루 멤버 → 지도에 표시 → 서로 응원"

페이스 조절:

"예상 완주 시간 확인 → 너무 빠르면 조절 → 목표 시간 달성"

💡 Tip: 배터리 절약

꼭 필요한 것만:

- 화면 밝기 낮추기

- 백그라운드 앱 종료

- 비행기 모드 + GPS만

응급 상황:

- 앱에서 "도움 요청"

- 자동으로 운영진 알림

민수의 한마디:

"대회는 혼자가 아니에요. 앱으로 서로 위치 보면서 응원하니까 더 힘이 나더라고요!"
