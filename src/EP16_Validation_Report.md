## ✅ Check 2. Character Voice (30/30)

### Blind Test: 대사만으로 캐릭터 구분 가능한가?

#### 1) 박민수 (복귀형 - 불안→자신감)
```
"다들... 보고 싶었어."
"응. 이제 뛸 수 있어."
"저... 정말 10km 뛸 수 있을까요? 2주 쉬었는데..."
"그래... 해보자."
'할 수 있어... 발목도 괜찮아...'
"했다!!!"
"유미야... 괜찮아?"
```
**Voice:** 불안 → 도전 → 성취, "..." 사용, 회복 과정 표현
**Tic:** 뒷목 긁기, 발목 신경 쓰기
**성장:** 부상 극복 → 10km 완주 성공

#### 2) 정유미 (좌절형)
```
"저... 정말 완주할 수 있을까요?"
'10km... 나 할 수 있을까...'
"네... 근데 이미 느린데..."
"형... 저 못 할 것 같아요..."
"형... 저 못 하겠어요..."
"못 해요... 저 안 돼요..."
"여러분... 죄송해요... 저 대회 못 나갈 것 같아요..."
"저 때문에 민폐잖아요..."
'...정말 할 수 있을까...'
```
**Voice:** 극도의 불안과 자책, "..." 다용, 계속되는 부정
**Tic:** 핸드폰 확인, 눈물
**위기:** EP12 성공 → EP16 좌절 (극적 반전)

#### 3) 한태수 (과묵형 → 감정 표현)
```
"...심박수 높아. ...천천히."
"...괜찮아. ...속도 아니야."
"...조금만 더."
"...할 수 있어."
"...괜찮아. ...오늘은 여기까지."
"...할 수 있어. ...진짜야."
"...유미야. ...내일 같이 뛰자. ...천천히."
"...민폐 아니야. ...가족이야."
```
**Voice:** 여전히 짧지만 **감정 깊이** 증가, "..." 필수
**Tic:** 물 건네기, 옆에서 묵묵히 동행
**성장:** 단순 침묵 → 핵심적 위로

#### 4) 이동현 (체계형)
```
"오늘은 대회 전 마지막 10km 테스트입니다."
"각자 자기 페이스대로 완주가 목표예요."
"Strava의 심박수와 페이스 분석 기능을 써보죠."
"심박수가 너무 높으면 속도 줄이고..."
```
**Voice:** 과학적, 체계적, 명확한 지시
**Tic:** 데이터 제시, 앱 시연

#### 5) 최순옥 (엄마형)
```
"유미야, 괜찮아?"
"쉬었다 가자."
"유미야, 우리 같이 하는 거제. 혼자 아니야."
```
**Voice:** 따뜻함, 사투리
**Tic:** 물 챙기기
