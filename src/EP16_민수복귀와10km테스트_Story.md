## 📱 The Tutorial Section

---

### 📊 Strava 심박수 & 페이스 분석 활용하기

대회를 앞두고 심박수와 페이스를 분석하면 안전하고 효율적인 러닝이 가능합니다.

---

#### 1️⃣ Logic: 왜 심박수와 페이스?

**심박수의 중요성:**
- 운동 강도 측정
- 과부하 방지
- 최적 유산소 구간 유지
- 부상 예방

**페이스의 중요성:**
- 속도 조절
- 에너지 배분
- 목표 시간 달성

---

#### 2️⃣ Stack: 필요한 것

- Strava 앱
- 스마트폰 (또는 스마트워치)
- GPS 활성화

---

#### 3️⃣ Guide: 4단계 활용법

**Step 1: 목표 심박수 계산**
```
최대 심박수 = 220 - 나이

예: 30세
최대 심박수 = 220 - 30 = 190 bpm

유산소 구간 = 최대 심박수의 70-80%
= 133-152 bpm
```

**Step 2: Strava 설정**
```
1. Strava 앱 → 설정
2. "센서" 선택
3. 심박수 센서 연결 (또는 폰 자체 측정)
4. "실시간 심박수 표시" 활성화
```

**Step 3: 런닝 중 확인**
```
화면에 표시:
- 현재 심박수 (bpm)
- 현재 페이스 (분/km)
- 거리
- 시간

심박수가 목표 구간 벗어나면:
- 높으면 → 속도 줄이기
- 낮으면 → 속도 올리기
```

**Step 4: 런닝 후 분석**
```
Strava 앱 → 완료된 활동
- 평균 심박수
- 최대 심박수
- 심박수 그래프 (구간별)
- 페이스 변화 그래프
```

---

#### 4️⃣ Prompt: 실전 활용

**10km 대회 준비:**
```
목표: 1시간 완주
필요 페이스: 6분/km
목표 심박수: 140-160 bpm

전략:
0-3km: 페이스 6분 30초 (여유)
3-7km: 페이스 6분 (유지)
7-10km: 페이스 5분 30초 (스퍼트)
```

**심박수 과부하 대처:**
```
심박수 170 이상 → 즉시 워킹
심박수 150 이하 → 다시 조깅
```

---

#### 💡 Tip: 효과적 활용

**처음엔 느리게:**
- 첫 3km는 목표 페이스보다 느리게
- 심박수 안정 후 속도 올리기

**중간 체크:**
- 5km마다 심박수 확인
- 과부하 시 즉시 조정

**마지막 스퍼트:**
- 8km 이후 여유 있으면 가속
- 심박수 무시하고 끝까지
