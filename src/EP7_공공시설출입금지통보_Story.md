### 🗺️ Google Maps로 완벽한 러닝 코스 찾기

장소가 막히면 어떻게 해야 할까요? 오늘은 **Google Maps**로 안전하고 좋은 러닝 코스를 찾는 법을 알아봅시다.

---

#### 1️⃣ Logic (원리): 왜 Google Maps인가?

Google Maps는 단순 지도가 아니라 **러닝 코스 설계 도구**입니다.

**3가지 핵심 기능:**
- **거리 측정:** 정확한 km 확인
- **경로 미리보기:** 골목, 조명, 경사 확인
- **리뷰 확인:** 다른 사람들의 평가

---

#### 2️⃣ Stack (도구): 무엇이 필요한가?

**필요:**
- 스마트폰 또는 PC
- Google Maps 앱 (무료)
- 인터넷 연결

**선택:**
- Strava (러닝 기록 앱)
- 종이 메모장 (코스 적기)

---

#### 3️⃣ Guide (실행): 어떻게 찾나?

**Step 1: Google Maps 열기**
- 앱 실행 또는 maps.google.com 접속

**Step 2: 검색어 입력**
```
"한강공원 러닝 코스"
"마포 한강 야간 러닝"
"안전한 러닝 코스 서울"
```

**Step 3: 거리 측정 도구 사용**
1. PC: 지도에서 우클릭 → "거리 측정"
2. 모바일: 지도 꾹 누르기 → "거리 측정"
3. 코스 따라 클릭 → 거리 자동 계산

**Step 4: 스트리트 뷰로 미리 보기**
- 노란 사람 아이콘 드래그
- 실제 거리처럼 둘러보기
- 조명, 보도, 경사 확인

**Step 5: 리뷰 확인**
- 근처 공원 리뷰 읽기
- "밤에 안전한가요?" 검색
- 별점 4.0 이상 추천

---

#### 4️⃣ Prompt (템플릿): 실전 활용

**상황 1: 안전한 야간 코스 찾기**
```
Google Maps 검색:
"[지역명] 한강공원 야간 조명"
"[지역명] 안전한 러닝 코스"

체크리스트:
□ 가로등 밝기 (스트리트 뷰)
□ 사람 유동 (리뷰 확인)
□ CCTV 유무 (공원 사이트)
```

**상황 2: 거리 정확히 재기**
```
PC 사용:
1. 출발점 우클릭 → "거리 측정"
2. 경로 따라 클릭
3. 도착점까지 거리 확인

결과: "총 5.2km" 표시
```

**상황 3: 경사 확인**
```
지형 레이어 활성화:
1. 왼쪽 상단 메뉴
2. "지형" 선택
3. 등고선으로 경사 확인
```
