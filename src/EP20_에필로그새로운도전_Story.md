**개인 맞춤형 계획:**
- 현재 수준 고려
- 목표 시간 설정
- 부상 예방 전략
- 유연한 조정 가능

**24/7 코치:**
- 언제든 질문 가능
- 즉각적인 피드백
- 무료로 이용

#### 2️⃣ Stack: 필요한 것

- ChatGPT 접속 (무료/유료)
- 현재 러닝 수준 파악
- 목표 대회 일정
- 주간 가능 횟수

#### 3️⃣ Guide: 효과적인 프롬프트

**Step 1: 현재 상태 공유**
```
"안녕! 나는 10km를 1시간 30분에 완주했어.

3개월 후 하프 마라톤 도전하려고 해. 
훈련 계획 짜줄래?"
```

**Step 2: 구체적 질문**
```
"Week 1-4는 어떤 거리를 뛰어야 해?"
"LSD 런은 어떻게 하는 거야?"
"페이스 런 vs 인터벌 차이는?"
"부상 예방 스트레칭 알려줘"
```

**Step 3: 계획 조정**
```
"Week 2에 무릎 통증이 있어. 
어떻게 조정하면 좋을까?"
```

**Step 4: 주간 점검**
```

```
