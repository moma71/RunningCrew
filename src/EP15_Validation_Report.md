## ✅ Check 2. Character Voice (30/30)

### Blind Test: 대사만으로 캐릭터 구분 가능한가?

#### 1) 김재희 (리더 대행 - 불안형)
```
"저... 오늘 5km만 가볍게 뛸게요."
"민수 형이 없으니까... 무리하지 말자고..."
"하지만 혹시라도 또 누가 다치면..."
"형, 우리 계속 준비할 거예요. 형 없어도."
"그리고 형이 돌아올 때까지 우리가 지킬게요. 크루를."
```
**Voice:** 불안 → 책임감 성장, "..." 다용, 조심스러운 말투
**Tic:** 뒤 돌아보기, 손 떨림
**성장 곡선:** 불안한 리더 → 단호한 약속

#### 2) 최순옥 (엄마형)
```
"재희야, 너 너무 긴장했다."
"리더는 완벽할 필요 없어. 그냥 다 같이 뛰면 되는 기다."
"민수는 민수고, 너는 너제. 네 방식대로 해봐."
"쑥, 소금, 당귀 넣었어. 혈액순환 좋아지고 붓기 빠진다."
"당연하제. 내 전문이다!"
"뭐가 고맙노. 우리 가족 아이가."
```
**Voice:** 사투리 ("~제", "~노"), 따뜻한 위로, 전문 지식 자랑
**Tic:** 허리에 손, 어깨 두드리기

#### 3) 박민수 (자책형)
```
"...그래. 잘했네..."
"...고마워. 정말 고마워."
"여러분... 제가 너무 걱정했네요. 여러분이 있어서 든든합니다."
```
**Voice:** 힘 없는 대답, "..." 다용, 감사 표현
**Tic:** 뒷목 긁기, 눈물

#### 4) 최진우 (에너지형)
```
"재희 누나, 너무 느린 거 아니에요?"
"형! 괜찮으세요?"
```
**Voice:** 직설적, 높은 에너지
**Tic:** "!"

#### 5) 한태수 (과묵형)
```
"...기다릴게."
```
**Voice:** 극도로 짧음, "..." 필수
**Tic:** 말 대신 메시지
