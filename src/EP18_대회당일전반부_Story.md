## 📱 The Tutorial Section

---

### 🏃 대회 실시간 트래킹 앱 활용하기

대규모 러닝 대회에서는 실시간 트래킹 앱으로 본인과 동료의 위치를 확인할 수 있습니다.

---

#### 1️⃣ Logic: 왜 실시간 트래킹?

**안전과 응원:**
- 참가자 위치 파악
- 가족/친구가 실시간 응원
- 응급 상황 대처
- 페이스 조절 참고

**주요 대회 앱:**
- 서울 마라톤: 공식 앱
- 춘천 마라톤: 실시간 순위
- 대부분 GPS 기반

---

#### 2️⃣ Stack: 필요한 것

- 대회 공식 앱 (사전 다운로드)
- 배번호 등록
- GPS 활성화
- 데이터 또는 Wi-Fi

---

#### 3️⃣ Guide: 3단계 활용

**Step 1: 사전 등록**
```
1. 대회 앱 다운로드
2. 배번호 입력
3. 프로필 설정
4. 위치 권한 허용
```

**Step 2: 출발 전 확인**
```
1. 앱 실행
2. GPS 신호 확인 (녹색)
3. "트래킹 시작" 활성화
4. 배터리 충분한지 확인
```

**Step 3: 런닝 중**
```
화면 표시:
- 현재 위치 (지도)
- 통과 지점 시간
- 예상 완주 시간
- 다른 참가자 위치 (선택)
```

---

#### 4️⃣ Prompt: 실전 활용

**가족/친구 응원:**
```
"배번호 12345 입력 →
실시간 위치 확인 →
5km 지점 도착 알림"
```

**크루 위치 확인:**
```
"같은 크루 멤버 →
지도에 표시 →
서로 응원"
```

**페이스 조절:**
```
"예상 완주 시간 확인 →
너무 빠르면 조절 →
목표 시간 달성"
```

---

#### 💡 Tip: 배터리 절약

**꼭 필요한 것만:**
- 화면 밝기 낮추기
- 백그라운드 앱 종료
- 비행기 모드 + GPS만

**응급 상황:**
- 앱에서 "도움 요청"
- 자동으로 운영진 알림
